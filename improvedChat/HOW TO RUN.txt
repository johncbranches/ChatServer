You will need JDK> 9 to run this program.

1. Run the server jar in your terminal:
(In the directory of the jar file) java -jar LocalNetworkChat-Server.jar

2. Run the client jar in another terminal window:
(In the directory of the jar file) java -jar LocalNetworkChat-Client.jar

3. For an enhanced experience, run at least one more client jar

4. Start chatting in the clients

5. Type /help and press enter (in one of the clients) for a small 
surprise.
